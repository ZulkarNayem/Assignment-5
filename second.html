Question-1: What are the different ways to select an element in the DOM?

There are different ways to select an element in the DOM. such as using getElementById, GetElementsByClassName, getElementsByTagName. We can also use querySelector and querySelectorAll.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

there are different between innerHTML, innerText and innerContent. they are: innerHTML takes text and HTML both, innerText takes only visible text inside the HTML tag and innerContent takes every text including hidden texts.

Question-3: What is event delegation in the DOM?

in DOM Event Delegation is a technique where an event listener is added to a parent element and that event also works for child elements using Event Bubbling. In short by using Event Delegation we access events like click, doubleClick, keyboard key press etc.

Question-4: What is event bubbling in the DOM?

Event Bubbling is a behavior in the DOM where an event starts from the child element and propagates up to it's parents oar grandparent. When an event occurs on a child element it first executes on that element and then bubbles up to it's parent and then grandparent and so on until it reaches the document object.

Question-5: How do you create, add, and remove elements using JavaScript?

In Javascript create, add, and remove elements are very simple. To create elements we use document.createElement(), to add them we use appendChild() and to remove we use removeChild(). These methods help dynamically update web pages.
